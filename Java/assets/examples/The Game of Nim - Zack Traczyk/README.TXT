PROJECT TITLE: The Game of Nim
PURPOSE OF PROJECT: To create the game of nim.
AUTHORS: Zack Traczyk
HOW TO PLAY: To when the Game of Nim you must make the computer take the marble.
             The game starts when the players are randomly given a pile of marbles,
             and the player or computer is randomly selected to go first. When it
             is your turn you can take anywhere from 1 to half the marbles.
